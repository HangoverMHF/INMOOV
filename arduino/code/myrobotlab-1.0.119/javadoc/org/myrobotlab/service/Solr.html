    public class Solr
extends Service
    SolrService - MyRobotLab This is an integration of Solr into MyRobotLab. Solr is the popular, blazing-fast, open source enterprise search platform built on Apache Lucene. This service exposes a the solrj client to be able to add documents and query a solr server that is running. For More info about Solr see http://lucene.apache.org/solr/
